The Periodic Table




What is the Periodic Table?

  • In the periodic table there are metals and non metals (classifications)
  • There are atomic numbers, atomic mass and symbol
  • There are 114 elements in the periodic table
  • The atomic number is the number of protons
  • The atomic mass is the mass of the neutrons and protons
  • The mass/nucleon number is the number of neutrons and protons
  • The rows of periodic table shows the number of rings/shells so it increases gradually
  • The rows are called the periods
  • The columns are called the groups
  • Elements in the same groups have the same number of electrons in the outermost shell
  • The mass number varies based on the element
